# Todo-app Simples

- Suporta as 4 operações básicas (Create, Read, Update, Delete)

## Estrutura

- Em src, ficam os arquivos .py principais do programa:
- src/
  - main.py - ponto inicial
  - utils.py - utilitários, pra receber nome, id, status, etc...
  - task_operations.py - manipular o banco de dados e as tarefas nele
 
- Em database, o banco de dados principal está localizado.
  - tarefas.db - onde as tarefas ficam armazenadas

## Requisitos:
 - Python 3.9+ (recomendado 3.11 pra cima)
 - Rich (pip install rich ou pip install -r requisitos.txt)
